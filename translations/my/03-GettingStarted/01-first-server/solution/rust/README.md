# ဒီနမူနာကို အလုပ်လုပ်စေခြင်း

## -1- လိုအပ်သော dependencies များကို ထည့်သွင်းပြီး ပရောဂျက်ကို တည်ဆောက်ပါ

```bash
cargo build
```

## -3- နမူနာကို အလုပ်လုပ်စေပါ

```bash
cargo run
```
